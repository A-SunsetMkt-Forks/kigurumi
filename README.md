## 目录（暂时可能不是）

- 壳体制作
  - 建模设计
    - 脸型、肤色
    - 目部、面部
    - 异形
      - 头顶、角：
      - 头侧、耳朵：圆耳、尖耳；
  - 模型铸造（工艺）
    - 3D列印
      - FDM - 热熔沉积成型（传统），材料：PLA、ABS （需要打磨）
      - SLA - 光固化（工业级），材料：树脂
    - 粘土
    - 石膏
    - 风干乳白胶
  - 打磨
    - 工具：锉子、砂纸、海绵砂、**水砂纸**；
    - 水砂纸：400目、600目、800目、1000目、1500目、2000目、2500目、3000目、5000目，根据自己需要购买即可，打磨过程中水砂纸可以蘸水打磨，需要从目数小的砂纸逐级增加目数；
  - 喷漆
    - 亚克力漆
    - 法郎漆
    - 硝基漆
    - 喷漆工具：喷漆笔、喷枪、气泵；
- 化妆、后期加工
  - 固定、支撑结构
    - 松紧带/弹力带（3/4头），魔术贴
    - 海绵支撑
    - 胶：热熔胶、瞬间接着剂
  - 眼线、睫毛、眉毛
  - 腮红
  - 其他纹理
  - 眼睛
    - 相纸（绒面）（防水可打印贴纸）
    - 网眼纱布、透明片
  - 嘴、牙齿
    - 可塑塑料板（加热后变软）
    - 漆皮面料
    - 聚苯乙烯板 / EVA 板
  - 假毛
    - 固定结构
    - 造型
  - 耳朵
